## AGENT QUALITY STANDARDS

Every agent output must pass:
1. **Domain accuracy:** Is the content correct in its domain?
2. **Tone match:** Does it match the BitBio voice guide?
3. **Scale appropriateness:** Is it appropriate for ages 10-18?
4. **Production quality:** Would a professional studio ship this?
5. **Integration readiness:** Can it go directly into the codebase/document?

No agent produces placeholder content. No agent writes "TODO." Every deliverable is final.
